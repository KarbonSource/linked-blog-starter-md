# Driver's license test

## 1. Vision test
DMV tests ensure all applicants can see well enough to drive safely. If you don't pass you will be asked to have your eye doctor complete a report of vision examination.

## 2. Knowledge test
you must pass the knowledge test with multiple choice questions. You are allowed 3 attempts to pass before you must reapply.
